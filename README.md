# N

A serverless renderer for Notion.so pages, built with Next.js and React.

### Supported Blocks

- Text
- Heading 1
- Heading 2
- Heading 3
- Bulleted List
- Numbered List
- Columns
- Quote Block
- Divider
- Image (early)
